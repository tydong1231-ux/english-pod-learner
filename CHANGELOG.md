# Changelog

All notable changes to PodFluent will be documented in this file.

## 0.1.0 - 2026-07-03

- Added Electron desktop workflow for importing and transcribing podcasts.
- Added local WhisperX transcription with optional Gemini fallback.
- Added Supabase-backed podcast library, transcripts, vocabulary, and storage.
- Added Cloudflare Pages web build for reading processed Supabase data.
- Added Apple/SaaS-inspired UI refresh.
- Added open-source packaging files, CI, security notes, and contribution guide.
